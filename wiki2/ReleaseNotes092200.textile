h2. Purpose

* changes in develop
* approved PRs

h2. New features

* larsim PR 67
** updates to use marley v1_2_0
** Eliminate the ugly hacks used to convert FHiCL into MARLEY&#39;s native JSON format. Instead, use a new class derived from fhicl::ParameterSetWalker to manage the conversion. Forego FHiCL validation of the generator settings in favor of letting MARLEY handle that itself. Adjust prodmarley.fcl so that it is experiment-agnostic.

h2. Bug fixes

* lardataalg PR 22
** Method in util::quantities::Quantity made const
** Due to an oversight, a few methods were non-const
* larwirecell PR 10
** make "ICARUS" namespace anonymous
** The module is for generic purposes, and the "ICARUS" namespace was unintentional.

h2. Updated dependencies

* marley v1_2_0
* nugen v1_12_00
* nusystematics v00_11_04
