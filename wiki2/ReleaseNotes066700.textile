* *build with e15 (gcc 6.4.0)*
* changes to develop
* approved feature branches

h2. New features

* larexamples feature/cerati_RecoProxyUsageExample

* caffe v1_0g
* mrb v1_14_00
** resolve #18838
** provide new changeQual command for use by code librarians
